## ğŸ“˜ Overview
The **Enrollment Management System (EMS)** is designed for educational institutions to efficiently manage and maintain essential academic data.
This system provides a user-friendly interface for performing **CRUD (Create, Read, Update, Delete)** operations on enrollment-related tables with built-in data validation to ensure consistency and accuracy across all records.

The **Table Maintenance Module** serves as the **administrative backbone** of the Enrollment Management System.

- Handle enrollment processes and scheduling
- Ensure data consistency across all related tables
